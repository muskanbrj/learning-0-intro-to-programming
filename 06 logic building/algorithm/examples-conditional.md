# Conditional

## Syntax

### if-end if
```
if <condition> then
[statements]
end if
```

### if-else-end if
```
if <condition> then
[statements]
else
[statements]
end if
```

### if-else if-else-end if (aka if-else ladder)
```
if <condition1> then
[statements]
else if <condition2> then
[statements]
else
[statements]
end if
```
